## Data Flow

1. **Data Ingestion**: WRDS data (CRSP, Compustat, IBES, TAQ) loaded via Data Loader
2. **Feature Engineering**: Raw data transformed into G1-G6 feature groups
3. **Encoding**: Dual-encoder (temporal + tabular) produces embeddings
4. **Ranking**: RankSCL generates similarity scores between stocks
5. **Filtering**: Liquidity constraints (Amihud, DArLiQ, spread, volume) applied
6. **Output**: Ranked list of liquid substitutes

## Key Modules

| Module | Responsibility |
|--------|----------------|
| `src/data/` | WRDS API integration, data loading, schema validation |
| `src/features/` | Feature engineering for G1-G6 groups (fundamentals, risk, price) |
| `src/models/` | Dual-encoder architecture (TCN/Transformer + FT-Transformer), RankSCL |
| `src/liquidity/` | Liquidity metrics calculation, constraint enforcement |
| `src/pipeline/` | End-to-end inference: query вҶ’ encode вҶ’ rank вҶ’ filter вҶ’ recommend |
| `src/config/` | Settings management, environment configuration |
